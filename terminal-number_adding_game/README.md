# Number Increment Game

Terminal based program that uses rand() to the advantage of the program.

The rules are simple.
- You are allowed to input numbers from 1 - 3
- The program is allowed to input numbers from 1 - 9
- You are not allowed to type any letters, negative integers, or decimal numbers. Program will count it as cheating.

This is a luck-based game, strategy involved.
Have fun, and don't quit early. It is a simple, and easy program to make work with when one is bored.
